# Release Plan

This repository is currently the project page for PHFD-Net while the paper is under
review. It shares paper-level information: method overview, quantitative and ablation
results, complexity profiling, figures, and dataset/evaluation protocol notes.

Upon acceptance, the authors plan to release:

- inference code;
- training code and configurations;
- pretrained checkpoints;
- evaluation scripts;
- re-parameterization conversion utilities (train-time to deploy-time);
- dataset preparation instructions.
